# Snake Game (Python + Pygame)

A classic Snake game built with Python and Pygame.  
Eat food, grow longer, and avoid crashing into walls or yourself.

## Features
- Smooth keyboard controls (Arrow keys or WASD)
- Pause/Resume with `P` key
- Score tracking
- Game over screen with restart (`R`)
- Grid-based movement with adjustable speed (fixed at normal)
